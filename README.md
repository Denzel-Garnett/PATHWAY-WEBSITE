🤖 IT Career Roadmap — Learning Guide
A comprehensive, interactive web-based roadmap designed to guide aspiring IT professionals from entry-level support roles to advanced infrastructure and network engineering positions. This tool aggregates the best free industry resources, certification paths, and hands-on lab environments into a single, cohesive curriculum.

🎯 Project Overview
The IT Career Roadmap serves as a "Mission Directive" for technical self-education. It breaks down the overwhelming world of Information Technology into four distinct, sequential phases, ensuring that learners build a solid foundation in hardware and networking before tackling complex cloud and automation concepts.

🚀 Key Features
Sequential Learning Tracks: Organized into four primary phases: Support, Infrastructure, Networking, and Modern IT.

Interactive UI: A tabbed interface allows users to drill down into specific skill sets (e.g., Active Directory, Kubernetes, Python for Network Engineers) without losing their place.

Resource Aggregation: Directly links to top-tier free educational content from Professor Messer, Cisco NetAcad, Microsoft Learn, and freeCodeCamp.

Certification Deep Dives: Detailed breakdowns of industry-standard certs (A+, Network+, Security+, CCNA, etc.), including exam costs, domain weights, and "pro-tips" for passing.

Hands-on Focus: Highlights free lab environments like Cisco Packet Tracer, ServiceNow Developer instances, and Microsoft Eval Center for practical experience.

🗺️ Roadmap Phases
Phase 01: Support (The Foundations)
The starting line for "Greenhorns." Focuses on the physical world, operating systems, and identity management.

Core Skills: Hardware, DNS/DHCP basics, Windows/macOS, Active Directory, and Ticketing (ITSM).

Phase 02: Infrastructure (Systems Engineering)
Moving from end-user issues to designing the systems that run the business.

Core Skills: Virtualization (VMware), Cloud (Azure/AWS), Storage Resilience (Veeam), and Monitoring (Datadog/Splunk).

Phase 03: Networking (The Plumbing)
Mastering the flow of data across global enterprises.

Core Skills: Subnetting, CCNA prep, VLANs, OSPF/BGP routing, and Firewall security (pfsense/Palo Alto).

Phase 04: Modern IT (Automation & AIOps)
The 2026 industry standard—bridging technical execution with AI-driven operations.

Core Skills: AI Literacy, Emotional Intelligence (Soft Skills), Kubernetes, and Infrastructure as Code (Terraform/Ansible).

🛠️ Technical Breakdown
The application is built using a "Single Page" philosophy for maximum speed and portability:

HTML5: Semantic structure for accessibility and SEO.

CSS3 (Modern UI): Uses CSS Variables for a dark-mode "Developer" aesthetic, Flexbox/Grid for responsiveness, and custom animations for tab switching.

Vanilla JavaScript: Lightweight logic for tab navigation and state management (no heavy frameworks required).

Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

🏅 Certification Guide Included
The roadmap includes a dedicated "Deep Dive" section for the following credentials:

CompTIA: A+, Network+, Security+, Linux+, and Cloud+.

Cisco: CCNA (200-301).

Note: Many of these certifications are recognized as substitutes for years of experience in specific professional applications (such as CUNY IT roles).

🏁 How to Use
Open sam.html in any modern web browser.

Start with Phase 01: Support.

Follow the Mission Directive at the top of each track.

Complete the "Level: Greenhorn" skills before advancing to "Runner" or "Elite" levels.
